Deep Learning Tournament
Опис проєкту

Deep Learning Tournament — це програмний комплекс для дослідження процесу навчання згорткових нейронних мереж на основі архітектури ResNet-18. Програма розроблена в межах бакалаврської роботи на тему «Застосування фрактальних фізико-інформованих нейронних мереж для аналізу даних».

Основною метою програмного комплексу є проведення експериментів із Fine-Tuning попередньо навченої моделі ResNet-18 та порівняння ефективності різних алгоритмів оптимізації під час класифікації зображень набору даних CIFAR-10.

Основні можливості
Навчання моделі ResNet-18 методом Fine-Tuning.
Використання попередньо навченої моделі ImageNet.
Підтримка оптимізаторів:
Adam;
SGD;

Налаштування гіперпараметрів навчання.
Відображення процесу навчання в реальному часі.
Побудова графіків Loss та Accuracy.
Автоматичне збереження натренованих моделей.
Турнірне порівняння декількох моделей.
Класифікація користувацьких зображень.

Структура програми

Програмний комплекс складається з трьох основних модулів:

1. Тренування моделі

Дозволяє запускати процес Fine-Tuning моделі ResNet-18 із вибраним оптимізатором та параметрами навчання.

2. Аналітика

Відображає графіки функції втрат (Loss) та точності класифікації (Accuracy) для навчальної та валідаційної вибірок.

3. Турнір моделей

Дає можливість порівнювати декілька натренованих моделей між собою на нових тестових зображеннях.

Набір даних

У роботі використовується датасет CIFAR-10.

Набір містить 60 000 кольорових зображень розміром 32×32 пікселі та 10 класів:

літак;
автомобіль;
птах;
кіт;
олень;
собака;
жаба;
кінь;
корабель;
вантажівка.
Результати дослідження

Під час проведення експериментів було виконано порівняння трьох алгоритмів оптимізації.

Отримані результати:

Оптимізатор	Accuracy
RMSprop	84.19 %
Adam	83.62 %
SGD	61.36 %

Найкращий результат продемонстрував алгоритм RMSprop.
